# 引物设计工具

NCBI Primer-BLAST 自动化工具，用于批量设计PCR引物。

## 主要功能

- 批量处理染色体坐标
- 自动坐标转换（hg19 → hg38）
- 参数模板管理
- 图形化界面操作

## 快速开始

### 安装包版本（推荐）

1. 运行 `build_installer.bat` 生成安装包
2. 安装后从开始菜单启动程序
3. 首次使用：工具 → 更新浏览器驱动
4. 输入坐标，设置参数，开始设计

### 源码运行

```bash
pip install -r requirements.txt
python run.py
```

## 坐标格式

```
chr1 123456
chr2 234567
X 345678
```

## 注意事项

- 首次使用需更新浏览器驱动（菜单 → 工具 → 更新浏览器驱动）
- 使用 hg19 坐标需下载转换文件（菜单 → 工具 → 下载坐标转换文件）
- 支持 Windows 7/8/10/11（64位）
